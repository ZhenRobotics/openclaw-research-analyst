1. 项目概述

1.1 项目定位

OpenClaw Research Analyst 是一个全市场覆盖的 AI 投资研究分析工具，整合 10+ 权威数据源，为个人投资者、量化研究者和财经从业者提供专业级的多维度投资分析服务。

1.2 核心特性

  • 全市场覆盖：支持美股、中国 A 股、香港股市、加密货币四大市场
  • 多维度分析：8 维度综合评分系统，覆盖基本面到技术面
  • 中文深度整合：集成 5 大中文平台（东方财富、新浪财经、财联社、腾讯财经、同花顺）
  • 完全开源免费：6063 行源代码可审查，核心功能无需任何 API 密钥
  • 本地部署：数据处理在本地完成，保护投资策略隐私

1.3 技术架构

组件 技术栈 说明
核心语言 Python 3.10+ 主要开发语言
包管理 uv 快速依赖管理
数据获取 yfinance, requests, BeautifulSoup 多源数据抓取
数据分析 pandas, numpy 数据处理与计算
代码规模 13 个脚本，6063 行代码 经过验证的生产级代码

2. 功能说明

2.1 美股分析 - 8 维度综合评分

评分维度

维度 权重 评估内容
盈利惊喜 30% EPS 实际值 vs 预期值，盈利质量分析
基本面 20% 市盈率、利润率、营收增长、债务水平
分析师情绪 20% 买入/持有/卖出评级、目标价变化
历史模式 10% 过往财报发布后的股价反应
市场背景 10% VIX 恐慌指数、SPY/QQQ 大盘趋势
板块表现 15% 相对板块的强度分析
技术动量 15% RSI、52 周区间位置、均线系统
市场情绪 10% 恐惧贪婪指数、空头比例、内部交易

信号生成

  • BUY（买入）：综合评分 ≥ 70 分
  • HOLD（持有）：综合评分 40-70 分
  • SELL（卖出）：综合评分 ≤ 40 分

风险检测

  • 盈利前警告（距离财报发布 < 14 天）
  • 超买警告（RSI > 70 且接近 52 周高点）
  • 后飙升风险（5 天内涨幅 > 15%）
  • 风险规避模式（避险资产 GLD/TLT/UUP 同步上涨）
  • 地缘政治风险（台湾、中国、俄罗斯、中东关键词）
  • 突发新闻警报（危机关键词扫描）

2.2 中国市场 - 5 大平台整合

数据源覆盖

平台 提供功能 脚本文件
东方财富 市场排行榜、板块分析、热门股票 cn_market_rankings.py
新浪财经 实时行情、A 股与港股报价 cn_stock_quotes.py
财联社 突发财经新闻、市场电报 cn_cls_telegraph.py
腾讯财经 资金流向分析、主力资金追踪 cn_tencent_moneyflow.py
同花顺 个股诊断、技术分析 cn_ths_diagnosis.py

功能特性

  • 多源验证：交叉对比 5 大平台数据，降低单一来源风险
  • 实时行情：A 股、港股实时报价（延迟 < 1 分钟）
  • 资金流向：主力资金、散户资金、北向资金追踪
  • 热点板块：自动识别市场热点和板块轮动
  • 个股诊断：技术面评分、买卖建议

2.3 加密货币分析

支持币种（前 20 大）

BTC, ETH, BNB, SOL, XRP, ADA, DOGE, AVAX, DOT, MATIC, LINK, ATOM, UNI, LTC, BCH, XLM, ALGO, VET, FIL, NEAR

分析维度

  • 市值与分类：市值排名、币种分类（Layer 1/DeFi/NFT 等）
  • BTC 相关性：30 天滚动相关系数
  • 技术动量：RSI、价格区间位置、趋势判断

2.4 投资管理工具

投资组合追踪

  • 创建多个投资组合
  • 记录持仓成本和数量
  • 实时计算盈亏和收益率
  • 集中度警告（单一资产占比 > 30%）

监控列表与警报

警报类型 触发条件 使用场景
目标价触发 当前价格 ≥ 目标价 止盈提醒
止损触发 当前价格 ≤ 止损价 风险控制
信号变化 BUY/HOLD/SELL 状态改变 策略调整

股息分析

  • 股息率：年度股息 / 当前价格
  • 派息比率：股息 / 每股收益（分类：安全/适中/高/不可持续）
  • 5 年增长率：股息 CAGR
  • 连续年数：连续增长的年数
  • 安全评分：0-100 综合评分
  • 收益评级：优秀/良好/适中/差

2.5 情报扫描系统

热点扫描器

多源病毒式趋势检测，数据来源：

  • CoinGecko Trending - 前 15 名热门币种
  • CoinGecko Movers - 最大涨跌幅
  • Google News - 财经与加密货币新闻
  • Yahoo Finance - 涨幅榜、跌幅榜、最活跃
  • Twitter/X - 社交媒体情绪（可选）

传闻检测器

早期信号识别，重点关注：

  • M&A 传闻：合并、收购、收购要约
  • 内部交易：CEO/董事买入/卖出
  • 分析师行动：升级、降级、目标价变化
  • Twitter 传言："据说..."、"有消息称..."、"传闻"
  • SEC 活动：调查、重要文件

3. 解决的问题

3.1 散户投资者的三大痛点

问题 1：数据分散，获取成本高

痛点描述：

  • 美股数据需要访问 Yahoo Finance
  • A 股需要翻墙访问多个中文网站
  • 加密货币又是另一套工具
  • 专业工具（如 Bloomberg）月费高达数千美元

解决方案：

  • 一个工具整合所有市场数据
  • 无需多平台切换
  • 所有数据源免费使用（无需 API 密钥）
  • 本地部署，无月费

问题 2：分析不全面，容易错过风险

痛点描述：

  • 散户通常只看价格和 K 线图
  • 忽略盈利质量、分析师看法
  • 忽略市场情绪、资金流向
  • 缺少地缘政治风险监控

解决方案：

  • 8 维度综合评分系统
  • 6 类风险检测（盈利前、超买、地缘政治等）
  • 全面覆盖基本面到技术面
  • 自动生成 BUY/HOLD/SELL 信号

问题 3：中文市场工具缺失

痛点描述：

  • 国外开源工具只支持美股
  • A 股/港股缺少好用的命令行工具
  • 中文数据源访问复杂

解决方案：

  • 深度集成 5 大中文数据源
  • 提供榜单、行情、新闻、资金流、诊断完整功能
  • 支持中英文双语

5. 使用场景

5.1 每日盘前准备（5 分钟）

# 中国市场热点
python3 scripts/cn_market_report.py

# 全球趋势扫描
python3 scripts/trend_scanner.py

# 检查监控列表警报
uv run scripts/watchlist_manager.py check

5.2 个股深度研究（3 分钟）

# 美股 8 维度分析
uv run scripts/stock_analyzer.py AAPL

# A 股实时行情
python3 scripts/cn_stock_quotes.py 600519

# 同花顺诊断
python3 scripts/cn_ths_diagnosis.py 600519

# 股息分析
uv run scripts/dividend_analyzer.py JNJ

5.3 寻找早期机会

# M&A 传闻、内部交易
python3 scripts/rumor_detector.py

# 病毒式热点扫描
python3 scripts/trend_scanner.py

# 财联社快讯
python3 scripts/cn_cls_telegraph.py

5.4 投资组合管理

# 创建投资组合
uv run scripts/portfolio_manager.py create "科技股组合"

# 添加持仓
uv run scripts/portfolio_manager.py add AAPL --quantity 100 --cost 150

# 查看盈亏
uv run scripts/portfolio_manager.py show

使用建议
