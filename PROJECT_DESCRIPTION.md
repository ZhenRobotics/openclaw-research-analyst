# OpenClaw Research Analyst - 项目简介

## 这个技能能干什么？

一个**全市场覆盖的 AI 研究分析工具**，为个人投资者提供专业级的多维度分析：

- **美股分析** — 8 维度综合评分（盈利、基本面、分析师、动量、情绪、板块、市场、历史），自动生成 BUY/HOLD/SELL 信号
- **中国市场** — 整合 5 大权威数据源（东方财富、新浪财经、财联社、腾讯财经、同花顺），覆盖 A 股与港股的排行榜、实时行情、资金流向、财经快讯、个股诊断
- **加密货币** — 前 20 大币种的市值、BTC 相关性、动量分析，识别热点趋势
- **投资管理** — 投资组合追踪、监控列表警报（目标价/止损/信号变化）、股息分析（收益率、派息比率、安全评分）
- **情报扫描** — 热点扫描器（多源病毒式趋势检测）、传闻检测器（M&A、内部交易、分析师行动的早期信号）

## 解决什么问题？

### 散户投资者的三大痛点

**1. 数据分散，获取成本高**
- **痛点**：美股用 Yahoo Finance，A 股要翻墙访问多个中文网站，加密货币又是另一套工具
- **解决方案**：一个工具整合所有市场数据，无需多平台切换，**所有数据源免费使用**（无需 API 密钥）

**2. 分析不全面，容易错过风险**
- **痛点**：只看价格和 K 线，忽略盈利质量、分析师看法、市场情绪、资金流向、地缘政治风险
- **解决方案**：8 维度综合评分 + 6 类风险检测（盈利前警告、超买、风险规避、突发新闻、地缘政治、后飙升风险），**全面覆盖基本面到技术面**

**3. 中文市场工具缺失**
- **痛点**：国外开源工具只支持美股，A 股/港股缺少好用的命令行分析工具
- **解决方案**：深度集成 5 大中文数据源，提供**榜单、行情、新闻、资金流、诊断**完整功能，支持中英文双语

## 效果如何？

### 数据准确性
- **价格数据**：与 Yahoo Finance 误差 <0.1%
- **财务指标**：直接来源于财报和权威数据源
- **中国市场**：5 大平台交叉验证，降低单一数据源风险

### 分析速度
- **标准模式**：60-120 秒完成 8 维度全面分析（包含 SEC 内部交易、突发新闻）
- **快速模式**：45-75 秒完成核心分析（跳过耗时的内部交易和新闻扫描）
- **中国市场报告**：30-60 秒生成多源整合报告

### 功能覆盖
- **13 个 Python 脚本**，6063 行经过验证的源代码
- **3 大市场**：美股、A 股/港股、加密货币
- **10+ 数据源**：Yahoo Finance, CoinGecko, SEC EDGAR, Google News, 东方财富, 新浪, 财联社, 腾讯, 同花顺, Twitter/X (可选)
- **零成本运行**：核心功能无需任何 API 密钥或付费订阅

### 实际应用场景

**场景 1：每日盘前准备**
```bash

python3 scripts/cn_market_report.py        # 中国市场热点
python3 scripts/trend_scanner.py           # 全球趋势扫描
uv run scripts/watchlist_manager.py check  # 检查监控列表警报
```

**场景 2：个股深度研究**
```bash
# 3 分钟完成 8 维度分析
uv run scripts/stock_analyzer.py AAPL                     # 美股
python3 scripts/cn_stock_quotes.py 600519                 # A 股行情
python3 scripts/cn_ths_diagnosis.py 600519                # 同花顺诊断
uv run scripts/dividend_analyzer.py JNJ                   # 股息分析
```

**场景 3：寻找早期机会**
```bash

python3 scripts/rumor_detector.py          # M&A 传闻、内部交易
python3 scripts/trend_scanner.py           # 病毒式热点扫描

```
