## 用户须知

### 安装前了解

#### 1. 核心功能立即可用（无需设置）

```bash

# 投资组合管理（无需凭证）

```

#### 2. 可选功能需要凭证（用户选择）

**Twitter/X 传闻扫描**:
- 需要 AUTH_TOKEN, CT0（浏览器 cookie）
- ⚠️ **会话 cookie 可访问您的账户 - 仅在信任此技能时提供**
- 没有凭证时技能自动降级使用 Google News

**飞书推送通知**:
- 需要 FEISHU_APP_ID, FEISHU_APP_SECRET, FEISHU_USER_OPEN_ID
- ✅ 官方 OAuth 2.0，仅机器人权限
- ✅ 提供设置向导: `python3 scripts/feishu_setup.py`
- 凭证保存到 `.env.feishu`（已被 git 忽略）

#### 3. 安装选项

```bash

npm install -g openclaw-research-analyst

# 方式 2: ClawHub 安装
clawhub install research-analyst

# 方式 3: 从源码安装
git clone https://github.com/ZhenRobotics/openclaw-research-analyst.git
cd openclaw-research-analyst
uv sync
```
