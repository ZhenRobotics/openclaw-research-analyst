# OpenClaw Research Analyst - 简短介绍

## 这是什么？

一个**全市场覆盖的 AI 投资研究工具**，整合 10+ 数据源，为个人投资者提供专业级的多维度分析。

## 能干什么？

- **美股**：8 维度综合评分（盈利/基本面/分析师/动量/情绪/板块/市场/历史），自动生成 BUY/HOLD/SELL 信号
- **A 股/港股**：整合 5 大中文平台（东方财富/新浪/财联社/腾讯/同花顺），提供排行榜、实时行情、资金流向、财经快讯、个股诊断
- **加密货币**：前 20 大币种的市值、BTC 相关性、动量分析
- **投资管理**：投资组合追踪、监控列表警报（目标价/止损/信号变化）、股息分析
- **情报扫描**：热点扫描器（多源趋势检测）、传闻检测器（M&A/内部交易/分析师行动早期信号）

## 解决什么问题？

1. **数据分散** → 一个工具整合美股/A 股/港股/加密货币，无需多平台切换
2. **分析不全面** → 8 维度评分 + 6 类风险检测，覆盖基本面到技术面
3. **中文市场工具缺失** → 深度集成 5 大中文数据源，支持中英双语

## 效果如何？

- **速度**：标准模式 60-120 秒，快速模式 45-75 秒
- **准确性**：价格数据误差 <0.1%，多数据源交叉验证
- **零成本**：核心功能完全免费（无需 API 密钥）
- **覆盖面**：13 个脚本，6063 行代码，10+ 数据源，3 大市场

## 典型使用场景

```bash
# 每日盘前 5 分钟
python3 scripts/cn_market_report.py        # 中国市场热点
python3 scripts/trend_scanner.py           # 全球趋势
uv run scripts/watchlist_manager.py check  # 检查警报

# 个股研究 3 分钟
uv run scripts/stock_analyzer.py AAPL     # 8 维度分析
python3 scripts/cn_ths_diagnosis.py 600519 # A 股诊断
```

## 核心优势

- **全市场** - 美股 + A股/港股 + 加密货币
- **多维度** - 8 维度评分 + 风险检测
- **中文深度** - 5 大平台深度集成
- **完全免费** - 核心功能无需付费
- **开源透明** - 6063 行代码可审查
